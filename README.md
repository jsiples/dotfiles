### Vim
**Add a plugin**

`git submodule add https://github.com/vim-airline/vim-airline.git .vim/pack/plugins/start/vim-airline`

**Update plugins**

`git submodule update --remote --merge`

**Delete plugins**

`git submodule deinit .vim/pack/plugins/start/vim-airline`
